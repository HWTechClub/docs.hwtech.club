Documentation Website for HWTechClub

### To get a local copy

1. Git clone the repo recursivily to get theme submodule
```shell
git clone --recurse-submodules -j8 https://github.com/HWTechClub/docs.hwtech.club.git
```
> **NOTE:** The '-j8' is an optional performance optimization that became available in version 2.8, and fetches up to 8 submodules at a time in parallel â€” see [man git-clone](https://git-scm.com/docs/git-clone)

2. Run Hugo local live server  
```shell
hugo server
```
3. Build site to {repo}/public/ folder 
```shell
hugo --gc --minify
```
